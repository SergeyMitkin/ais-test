Доски сырые - это один из самых востребованных и покупаемых материалов. Их нарезают из бревен, обрабатывают со всех сторон. Гладкая и ровная поверхность без коры делает такой материал весьма удобным в работе.

Где же используютсырые материалы?
На самом деле спектр использования подобных материалов достаточно широкий.
Его применяют в строительстве, в некоторых случаях покрывая антисептическим средством

Лучше всего он подходит для наружного применения:

  • для наружной облицовки сооружений;
  • для кровли домов и коттеджей;
  • для постройки внешних стен зданий;
  • для строительных лесов;
  • для опалубок при закладке фундамента;
  • для деревянных палет;
